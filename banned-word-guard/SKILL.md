# 违禁词排查与替换工具

## Overview

这个工具用于在公众号文章**定稿前**做最后一道安全扫描：

1. 扫描标题和正文中的违禁词
2. 发现违禁词后，给出**安全替换建议**
3. 自动替换为合规表达，输出清洁版

它覆盖微信平台审核和《广告法》两套红线，同时也包含视频号健康内容的专项红线。

如需完整词库，读取 `references/word-lists.md`。

## 适用任务

- 文章写完后、发布前的安全扫描
- 标题敏感词排查
- 健康养生类内容的红线自查
- 广告法违禁词自查
- 电商/带货类内容的功效词自查

## 扫描范围

### 一、绝对化用语（广告法红线）

与"最"有关的：最、最好、最高级、最佳、最具、最爱、最优、最优秀、最科学、最先进、最新、最低、最低价、最便宜、最符合、最舒适、最时尚、最流行、最受欢迎、史上最低价、全网最低价、最高荣誉、最大、最大程度、最奢侈

与"级"有关的：极致、极品、极佳、终极、国家级、国际级、世界级、全球级、特级、冠级、宇宙级

与"顶/鼎"有关的：顶尖、顶端、顶级、鼎冠、问鼎、TOP级、顶流、顶配

与"领"有关的：遥遥领先、领袖品牌、世界领先、领先、领军者

与"绝"有关的：绝版、绝佳、绝对超值、空前绝后

与"第一"有关的：第一、NO.1、TOP.1、全国第一、中国第一、全网第一、销量第一、排名第一

与"首"有关的：首个、首选、首款、首家、全国首家、全球首发、首席、首次、首创、首屈一指

其他绝对化用语：销冠、销量冠军、空前、无双、完美、100%、百分之百、万能、永久、独家、免检、王牌、无敌、唯一、独一无二、一流、金牌、MVP、率先、不二之选、全能、尖端

### 二、医疗健康红线（视频号/公众号高压线）

1. 医疗行为类（最致命）
   - 直接诊断疾病："你这是肝不好""湿气重"
   - 明确治疗方案："这样吃就能治好"
   - 替代医疗："不用去医院""比吃药还管用"
   - 指导用药："这个比某某药更好"

2. 疗效承诺类（100%违规）
   - 绝对化表达："一定有效""根治""彻底解决""立刻见效"
   - 夸大效果："三天改善""7天逆转""一个月年轻10岁"
   - 对比贬低医疗："医院都治不好，这个可以"

3. 疾病敏感词触发类
   - 明确严重疾病名称：癌症、肿瘤、心梗、脑梗、糖尿病、抑郁症
   - 症状引导焦虑："如果你有这3个表现，你已经很危险了"
   - 自测+对号入座："符合2条以上说明你身体出问题了"

4. 制造恐慌/焦虑（清朗重点打击）
   - 死亡威胁："再这样下去你会出大事"
   - 放大风险："90%的人都在慢性自杀"
   - 利用热点恐吓："某某突然倒下，你还不警惕？"

5. 食疗=治疗（高发踩坑）
   - "吃XX可以治XX病"
   - "某食物抗癌/降三高/修复器官"
   - "比药还有效的食物"

6. 身份冒充/权威背书
   - 冒充医生/专家
   - 暗示医疗资质："从业20年告诉你"
   - 虚假权威："专家都在用""医院内部方法"

### 三、医疗行业敏感词（非资质不可用）

药方、药用、药物、中药、医疗、医治、治疗、消炎、抗炎、活血、解毒、抗敏、中医、西医、诊治、就诊、康复、体检、化验、开方、抗病毒、手术、注射、化疗、理疗、退烧、止痛、止咳、西药、处方、祛瘀、清热解毒、补肾、祛风、防癌、抗癌、降血压、平衡荷尔蒙、调节内分泌、延缓更年期

### 四、食品功效敏感词（非保健食品不可用）

增强免疫力、降血脂、降血糖、抗氧化、改善记忆、提高记忆力、缓解视疲劳、促进排铅、辅助降血压、改善睡眠、促进泌乳、缓解疲劳、减肥瘦身、减脂、增加骨密度、改善贫血、护肝、养肝、祛痘、祛斑、补水、控油、调节肠道菌群、促进消化、通便、润肠

### 五、化妆品功效敏感词（非特殊化妆品不可用）

美白、祛斑、淡斑、亮白、变白、焕白、嫩白、增白、育发、促进毛发生长、固发、防脱发、除臭、去狐臭

### 六、夸张禁用语

至臻、臻品、非此莫属、无人能及、翘楚之作、不可复制、绝无仅有、前所未有、无与伦比、稀世珍宝、史无前例、万人疯抢、全民疯抢、抢疯了、亏本价、抢爆、卖疯了、随时涨价、神丹、祖传秘方、立竿见影、起死回生、返老还童、无法超越、业界公认

### 七、国家形象相关用语

国家免检、中国驰名、驰名商标、国家领导人推荐、国家机关推荐、国宴、国酒

### 八、承诺保证用语

无效退款、零风险、特效、全效、强效、奇效、速效、神效、有效率、保证、根治、100%有效、无副作用、无风险、彻底消除、完全清除、不反弹、不复发

### 九、违规话术结构（算法直接识别）

- "医生不会告诉你的…"
- "赶紧收藏，否则来不及"
- "不看后悔一辈子"

## Workflow

### 场景A：完整扫描（默认）

当用户给出一篇文章（标题+正文），要求排查违禁词时：

1. 逐条对照上述九类词库扫描
2. 分别检查：标题、正文
3. 对每个命中的违禁词：
   - 标出原文和位置（标题 or 正文第N段）
   - 标注属于哪一类风险
   - 给出 1 到 3 个安全替换建议
4. 输出替换后的清洁版全文

### 场景B：只查不改

当用户说"帮我扫一遍，先别改"时：

1. 扫描全文
2. 只输出命中的违禁词清单和风险类别
3. 不做替换，等用户确认后再改

### 场景C：只查标题

当用户说"帮我查一下标题"时：

1. 只扫描标题
2. 标出命中的违禁词
3. 给出替换建议

## 替换原则

### 原则1：替换后语义不变

替换词必须保持原文想表达的意思，不能因为替换而丢失信息或改变立场。

### 原则2：替换后语气自然

替换词要符合文章的人设和语气，不能替换后反而像机器人。

### 原则3：能改写就改写，能删就删

如果一个违禁词不是核心表达，优先删掉或换一种说法，而不是只替换一个词。

示例：
- "最好的方法" → "比较靠谱的方法" / "我试过最管用的方式"
- "100%有效" → "亲测有帮助" / "身边很多人反馈确实不错"
- "治疗失眠" → "改善睡眠" / "帮助入睡"
- "根治" → "从根本上改善" / "长期调理"
- "第一个告诉你" → "很早就在提醒你" / "很多人没意识到"

### 原则4：健康类特殊处理

健康养生内容的违禁词最容易踩坑，替换时注意：

- "治" → "改善" / "调理" / "养护"
- "药" → "食材" / "食材搭配"
- "病" → "不适" / "状况" / "小问题"
- "医院都治不好" → 删除整句或改为"很多人折腾了很久"
- "抗癌食物" → "对身体有益的食物" / "值得常吃的食材"

|---|------|------|----------|----------|

```

|---|------|------|----------|----------|

```

## 与 Pipeline 的衔接

这个 skill 在 `wechat-article-pipeline` 中作为**最后一步**自动触发：

```
写稿 → 降AI → 违禁词排查 → 发布素材
```

也可以单独调用，不依赖 pipeline。

## 注意事项

1. 扫描时不只是精确匹配，也要关注**语义层面**的违规。比如"效果堪比手术"虽然没有出现"治疗"，但暗示了医疗效果
2. 有些词在特定语境下是安全的，比如"这是她第一次做饭"中的"第一次"不属于违禁。需要根据上下文判断
3. 健康类内容需要格外严格，因为视频号和公众号的健康审核是独立且更严的
4. 替换后要重新读一遍，确保语句通顺、不生硬
5. 如果一篇文章命中超过 5 处高风险违禁词（医疗行为类、疗效承诺类），建议直接重写相关段落，而不是逐词替换
