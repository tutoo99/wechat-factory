# 杂志柔感预览

这套风格更像一篇专栏，而不是一份功能说明书。它希望读者读起来更轻一点，也更愿意停留。

## 先让情绪落下来

有些文章不是用来教人做事的，而是用来陪人想明白一点事情。那种时候，排版也不应该太硬。

### 一点区别

柔和不是幼稚，克制也不是冷淡。它只是让内容和情绪的速度更接近。

> 如果一篇情绪文从视觉到语言都太像文档，它很难留下温度。

- 留一点呼吸感
- 留一点停顿感
- 留一点“专栏感”

1. 先把读者带进去
2. 再慢慢展开判断
3. 最后给一句能带走的话

```text
不是所有文章都要显得效率很高。
有些文章更重要的是：让人愿意读完。
```

| 项目 | 气质 |
|------|------|
| 标题块 | 更面状 |
| 分隔线 | 更轻 |
| 引用块 | 更柔和 |
