# 粥左罗极简预览

这是一段导语，和正文同字号，不做视觉区分。整体极简，不依赖任何背景色、卡片、边框来制造视觉层次，让文字本身说话。

## 大标题：纯加粗无装饰

他打太平天国时，屡战屡败，甚至投江自杀，最艰难的时候，所有人都劝他放弃。

但他没有。就是遇到困难，不要退缩，不要放弃，只需要确保方向是正确的就好。

**今天分享的这10句话，字字珠玑，句句见血，句句入心。**

### 子标题：加粗放大

不是天才，考了7次才中秀才；脾气暴躁，得罪了无数人；出身寒门，没有任何背景。

但就是这样一个普通人，最后成了晚清四大名臣之首。

**不要总问"我什么时候能成功"，而要在心里面清晰自己的目标，然后问"我今天做了什么"和目标有关系的事情。**

不要总想着"如果当初我怎样怎样"，也不要总想着"以后我要怎样怎样"，而是把心放在当下，专注眼前的事情。

用曾国藩的思维去做事，你会发现很多事曾国藩已经给出了解决方案。
