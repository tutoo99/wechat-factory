# 情感暖调预览

这是一段用于验证 `emotion-warm` 主题的人味导语。它应该比正文略大一点，更像公众号里那种“先把你拉进来”的开场。

## 先把话说在前面

有些情绪文，不是靠大道理打动人，而是靠一句你也经历过的话把你轻轻带进去。

### 一个小提醒

如果一篇文章从头到尾都像会议纪要，它就算道理没错，也很难让人愿意读完。

> 人真正愿意收藏一篇文章，很多时候不是因为它更完整，而是因为它刚好替自己把心里那句话说出来了。

- 列表一：保留呼吸感
- 列表二：保留留白
- 列表三：保留真实口气

1. 先让读者停下来
2. 再让读者点头
3. 最后再给一句带得走的话

这是一段带 `行内代码` 的正文，用来验证轻量代码样式也不会特别突兀。

```text
真正好看的情感号排版，不是堆很多花样，
而是让读者一直愿意往下滑。
```

| 项目 | 说明 |
|------|------|
| 字号 | 稍大一些 |
| 行距 | 呼吸感更强 |
| 语气 | 更亲近 |
